
The project was create using Eclipse IDE.

Instructions to compile and execute the program are below:

1. Compile the .java files in the "src" folder using the below command in the terminal window:
   javac *.java
   
2. The program execution syntax is as follows:
   a) To test insertion sort type below command in the terminal window
	  java SortList i number
   b) To test quick sort type below command in the terminal window
      java SortList q number
   Note: number - has to be a positive integer. 
   		 i denotes insertion sort and q denotes quick sort algorithm.
   		 And execution will only work in the directory containing the .class files
